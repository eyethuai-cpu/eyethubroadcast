# Eyethu Pro Radio Studio

Professional broadcast radio studio PWA — single-file, runs fully offline.

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your studio will be live at `https://<username>.github.io/<repo>/`

## Features
- Dual decks (A/B) with waveform display
- Mic with DSP chain (HPF → AGC → Compressor → EQ → Limiter)
- Per-channel audio device routing (Settings tab)
- Cartwall, Show Scheduler, Auto-DJ
- Fully offline / PWA installable
